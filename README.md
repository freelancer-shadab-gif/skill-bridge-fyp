# skill-bridge-fyp
🚀 Skill Bridge – AI-Powered Career Guidance Platform
Skill Bridge is a modern web-based career guidance system designed to help students and beginners choose the right career path based on their skills, interests, education, and strengths.
It provides AI-inspired career recommendations, structured learning roadmaps, certification guidance, and curated learning resources in one platform.

🎯 Problem Statement
Many students struggle to choose the right career due to:
•	Lack of proper guidance
•	No clear learning roadmap
•	Confusion about skills and career scope
Skill Bridge solves this problem by providing personalized career suggestions and structured learning paths.
💡 Features
•	🧠 Career Assessment System (Skill + Interest Matching)
•	📊 AI-Inspired Career Recommendation Engine
•	🛣️ Step-by-Step Learning Roadmaps
•	🎓 Career Certifications Guidance
•	🎥 YouTube Learning Hub Integration
•	💬 AI Chatbot Assistant (Rule-based)
•	🔐 Simple Authentication (Local Storage)
•	📱 Fully Responsive UI (Mobile + Desktop)

🛠️ Technologies Used
•	HTML5
•	Tailwind CSS
•	JavaScript (ES6)
•	Local Storage (Browser-based storage)
•	Lucide Icons

⚙️ How It Works
1.	User enters:
o	Education
o	Skills
o	Interests
o	Strengths
2.	System analyzes input using keyword matching logic.
3.	Career match score is generated.
4.	User receives:
o	Top career recommendations
o	Skill requirements
o	Certifications
o	Learning roadmap
